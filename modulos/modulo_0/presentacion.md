# Presentación del Módulo 0: Preparación y Descubrimiento Artístico

## Descripción General

Este módulo introductorio está diseñado para romper el hielo y eliminar el "miedo a la hoja en blanco". No buscamos obras maestras, sino establecer una relación positiva con el dibujo. El enfoque principal es la exploración sensorial de los materiales y la liberación motriz a través del garabato y el juego. Es la base emocional y física para todo el curso.

## Objetivos de Aprendizaje (Competencias)

1. **Confianza Creativa**: Comprender que el error es parte del juego y disfrutar el proceso sin autojuzgarse.
2. **Exploración Material**: Identificar y probar las posibilidades de lápices, crayones y marcadores.
3. **Soltura Motriz**: Realizar trazos amplios y libres, moviendo el brazo completo y no solo la muñeca.
4. **Lenguaje Visual**: Entender que diferentes líneas (rápidas, lentas, gruesas) comunican diferentes sensaciones.

## Rúbricas de Evaluación Sugeridas

| Criterio | Excelente (10) | Bueno (8) | En Desarrollo (6) |
| :--- | :--- | :--- | :--- |
| **Actitud y Participación** | Muestra entusiasmo visible, prueba todos los ejercicios sin miedo y se divierte. | Realiza las actividades con interés, aunque duda ocasionalmente antes de trazar. | Necesita constante ánimo para empezar; muestra inseguridad o miedo a "hacerlo mal". |
| **Exploración de Herramientas** | Usa diferentes materiales (lápiz, crayón) variando presión y agarre. Experimenta activamente. | Usa los materiales correctamente pero tiende a quedarse con uno solo o usar siempre la misma presión. | Usa los materiales con timidez o rigidez excesiva (aprieta demasiado siempre). |
| **Calidad del Trazo (Soltura)** | Trazos fluidos y seguros. Ocupa gran parte de la hoja. Se nota el movimiento del brazo. | Trazos controlados. Ocupa un espacio moderado. A veces el trazo es "peludo" o inseguro. | Trazos muy pequeños, rígidos o temblorosos. Dibuja solo en una esquina o con miedo. |

## Recomendaciones Didácticas para el Instructor/Padre

* **Prohibido Borrar**: En esta etapa, esconde la goma de borrar. Fomenta que cada línea, "correcta" o "incorrecta", se quede en el papel.
* **Música de Fondo**: Usa música rápida para ejercicios de garabatos y música lenta para líneas suaves. Ayuda a la motricidad.
* **Modelo a Seguir**: Dibuja tú también (y comete errores a propósito). Si el niño ve que el adulto se divierte garabateando, perderá el miedo.
* **Postura**: Vigila que no se encorven demasiado sobre la hoja. El "Agarre de Varita" (desde atrás del lápiz) es ideal para estos ejercicios iniciales.
