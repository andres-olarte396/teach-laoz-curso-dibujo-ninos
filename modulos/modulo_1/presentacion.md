# Presentación del Módulo 1. Fundamentos de Formas Básicas

## Descripción General

En este módulo, los estudiantes descubren que "todo dibujo complejo nace de formas simples". Aprendemos el alfabeto visual: el Círculo, el Cuadrado y el Triángulo. El objetivo es desmitificar el dibujo, demostrando que si puedes dibujar estas tres figuras, puedes dibujar cualquier cosa (desde un robot hasta un gato).

## Objetivos de Aprendizaje (Competencias)

1. **Reconocimiento Geométrico**: Identificar formas básicas en objetos del mundo real.
2. **Control de Trazo Cerrado**: Dibujar figuras cerradas (donde el inicio y el fin se encuentran limpiamente).
3. **Construcción Modular**: Combinar formas simples para crear objetos complejos (Ej: Cuadrado + Triángulo = Casa).
4. **Uso de la Herramienta**: Aprender a mover el hombro para círculos grandes y la muñeca para detalles pequeños.

## Rúbricas de Evaluación Sugeridas

| Criterio | Excelente (10) | Bueno (8) | En Desarrollo (6) |
| :--- | :--- | :--- | :--- |
| **Cierre de Formas** | Las figuras están bien cerradas; no hay espacios abiertos ni líneas que se cruzan excesivamente. | La mayoría de las figuras cierran bien, con pequeños errores de precisión. | Las figuras quedan abiertas o las líneas se cruzan demasiado, perdiendo la forma. |
| **Identidad Geométrica** | El cuadrado parece cuadrado (lados rectos), el círculo es redondo (no huevo). Se distinguen claramente. | Las formas son reconocibles, aunque los círculos pueden ser ovalados o los cuadrados algo torcidos. | Cuesta distinguir entre un círculo y un cuadrado; trazos muy deformados. |
| **Creatividad Combinatoria** | Logra construir objetos reconocibles (casa, robot) usando las formas. Añade detalles propios. | Construye los objetos solicitados siguiendo las instrucciones básicas. | Tiene dificultad para ensamblar las formas juntas (ej: el techo flota separado de la casa). |

## Recomendaciones Didácticas para el Instructor/Padre

* **Técnica del Fantasma**: Enseña al niño a practicar el movimiento en el aire (sobre el papel sin tocarlo) antes de trazar.
* **Rotación del Papel**: Está permitido girar la hoja si les resulta más cómodo para trazar ciertas líneas.
* **Analogías**: Usa nombres divertidos (Círculo = Pizza/Moneda, Triángulo = Pizza rebanada/Montaña, Cuadrado = Caja/Ventana).
* **Corrección Positiva**: Si el cuadrado sale chueco, di "¡Ese es un cuadrado bailando! Ahora hagamos uno que esté quieto".
